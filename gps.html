GPS LED Watch

Problem

I saw a cool project while was scrolling through Adafruit's website and wanted to makethis GPS watch during my spare time.


Objective

To follow the Adafruit’s tutorial on the FLORA GPS watch and make a few of my own modifications.

Skills Employed

  • Arduino
  • Neopixel LED Ring
  • GPS Module
Figure 1. GPS watch in GPS Clock Mode - Blue for Minutes, Red for Hour

Approach

  • Followed the tutorial but also made a few more modifications:
    • Created a flashlight mode (Figure 4)
    • Changed the colors of the shown LEDs
  • Went around the neighborhood and tested out the GPS module (as shown in Figure 3.)

Figure 2. GPS Watch - Compass Mode

Figure 3. GPS Watch - GPS Mode
Figure 4. GPS Watch - Flashlight Mode

Results

Though there are definitely better watches and GPS devices, this project was a fun way to experiment with the GPS module, neopixel ring, the FLORA board, and an accelerometer.
