wooden end table

Problem

I saw an end table on Amazon but I wanted to build it myself and have my own woodworking project (plus an end table for my living room would be a nice addition to the space).


Objective

To fabricate a wooden end table (similar to the design in Figure 1).

Skills Employed

  • Onshape 3D CAD model
  • Chop Saw
  • Vertical Bandsaw
  • Joiner
  • Planar
  • Belt Sander
  • Joining Biscuits
Figure 1. Amazon End Table
Link to Amazon End Table

Approach

  • Getting funding for resources:
    • To create a bill of materials (BOM) and an Onshape 3D CAD model (of the overall design, dimensions, and assembly) to apply for the Engineering Innovation Fund at Boston University.
  • Making Table Legs (Figure 3.)
    • After receiving all of my parts, I first drew out the dimensions on the wood I received.
    • Since the makerspace did not have any table saws, I cut the shape of the legs using the chop/miter saw and the vertical band saw. Then I sanded down any edges.
  • Making Rattan Book Rest
    • Marked the dimensions on the wood
    • Used the bandsaw to cut out the parts
    • Sanded the edges
  • Making Table Top
    • Used the joiner and planar to decrease the height of the wood.
    • Marked the dimensions on the wood
    • Still in progress! Come back later for more information!
